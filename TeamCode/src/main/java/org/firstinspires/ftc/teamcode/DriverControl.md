# Gamepad 1 (Drivetrain)
- Left Stick
  - Drive the robot in the direction of the stick
- Right Bumper
  - Slowmode
- Triggers
  - Turn the robot

# Gamepad 2 (Manipulation)
- Manual Mode
  - Lift
    - Y for high
    - A for mid
    - B for low
    - X for intake
  - Right bumper to toggle claw
  - Left Stick to aim

# All
- Back Button:
  - Recalibrate orientation
  - Must be aligned in the bottom left corner
